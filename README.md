# Compilation Instructions

```bash
git clone --recursive https://github.com/NYUCG2016/Assignment1
cd cg
cd Assignment_1
mkdir build
cd build
cmake ../
make
```

If you are looking for an IDE, I suggest to use [CLion](https://www.jetbrains.com/clion/).
